# Contributing to Fast React Pizza Co.

Thank you for your interest in contributing! Here's how you can help:

## Getting Started

1. Fork the repository
2. Clone your fork: `git clone https://github.com/yourusername/03-pizza-menu.git`
3. Create a feature branch: `git checkout -b feature/your-feature-name`
4. Install dependencies: `npm install`
5. Start development: `npm start`

## Development Guidelines

- Write clean, readable code
- Follow React best practices
- Use meaningful variable and function names
- Test your changes before submitting
- Add comments for complex logic

## Submitting Changes

1. Commit your changes: `git commit -m "Brief description of changes"`
2. Push to your fork: `git push origin feature/your-feature-name`
3. Create a Pull Request with a clear description

## Code Style

- Use functional components
- Use React Hooks for state management
- Keep components focused and reusable
- Add CSS comments for styling logic

## Questions?

Feel free to open an issue for any questions or suggestions!
